CSS Layout Basics

Normalize vs. CSS Reset

  1. Normalize.css preserves useful defaults, so you don't have to redeclare styles for all the common typographic elements.
  2. Normalize.css corrects common bugs.
  3. Normalize.css doesn't clutter your DevTools.
  4. It has great inline, modular documentation so you know where to look for stuff.

Layout Wrapper or Container

Wrapper or container holds the content and centers it on the page. We can use 'body' as a container but it's best to use the 'wrapper' or 'container'. That way if we need to play items outside of the wrapper, we can.

Why vertical margins collapse

Initial gap b/w header and top of page come from the h1 top margin which is collapsing because there is no padding for the wrapper. When we add the top/bottom padding to the header, it fixes the problem and there is no gap. Collapsing of the top/bottom margins can happen between adjasent divs, so the larger margin of the div wins. Horizontal margins don't collapse.
